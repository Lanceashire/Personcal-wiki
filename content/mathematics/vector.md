# 向量

## 条目概览

向量是按顺序排列的一组数。几何上，它可以表示带方向的位移；在数据处理中，它也可以表示一个样本的多项特征。两种理解并不冲突：坐标就是向量在选定基底下的数值表示。

## 为什么需要它

单个标量只能描述一个量。现实问题通常同时包含多个维度，例如房屋的面积、房龄和卧室数。把它们写成向量后，就可以用统一的代数运算处理距离、方向、线性组合与相似度。

## 直观理解

二维向量 $\mathbf{x}=(3,4)$ 可以看作从原点走 3 个单位的水平方向，再走 4 个单位的竖直方向。它的长度为：

$$
\|\mathbf{x}\|_2=\sqrt{3^2+4^2}=5
$$

## 正式定义

一个 $n$ 维实向量写作：

$$
\mathbf{x}=(x_1,x_2,\ldots,x_n)^\mathsf{T}\in\mathbb{R}^n
$$

其中 $x_i$ 是第 $i$ 个分量。向量加法和数乘逐分量进行。

## 核心组成与工作原理

| 操作 | 形式 | 含义 |
| --- | --- | --- |
| 加法 | $\mathbf{x}+\mathbf{y}$ | 合成两个位移或特征变化 |
| 数乘 | $a\mathbf{x}$ | 缩放长度，负数还会反转方向 |
| 点积 | $\mathbf{x}^\mathsf{T}\mathbf{y}$ | 衡量同向程度并形成加权和 |
| 范数 | $\|\mathbf{x}\|$ | 衡量向量大小 |

## 完整计算示例

设 $\mathbf{x}=(1,2)$，$\mathbf{y}=(3,-1)$：

$$
\mathbf{x}+\mathbf{y}=(4,1),\qquad
\mathbf{x}^\mathsf{T}\mathbf{y}=1\times3+2\times(-1)=1
$$

## 代码或工程中的表现

```python
import numpy as np

x = np.array([1.0, 2.0])
y = np.array([3.0, -1.0])
print(x + y)       # [4. 1.]
print(x @ y)       # 1.0
```

## 与相似概念的区别

- 标量只有大小；向量有多个分量，并可解释为大小与方向。
- 矩阵是按行列组织的二维数组，可以表示线性变换；向量常作为矩阵的输入或输出。
- 坐标依赖所选基底，而抽象向量本身不依赖某一组具体坐标。

## 常见误区

- 向量不一定必须画成箭头；特征向量同样是向量。
- 向量长度不等于分量之和。
- 点积的结果是标量，不是向量。

## 边界与限制

不同量纲的特征直接组成向量时，大数值尺度可能支配距离和点积，因此建模前常需标准化。向量也只表达数值与顺序，不会自动保存每个分量的语义。
