# 线性回归

## 条目概览

线性回归用输入特征的加权和预测连续数值。它既是可直接使用的回归模型，也是理解损失函数、参数估计、正则化和神经网络线性层的重要起点。

## 基本信息

| 属性 | 内容 |
| --- | --- |
| 学习范式 | 监督学习 |
| 预测目标 | 连续变量 |
| 模型形态 | 特征的线性组合 |
| 常用损失 | [[均方误差\|MSE]] |
| 常用求解 | 正规方程、[[梯度下降]] |

## 为什么需要它

许多任务需要估计一个连续量，例如根据面积预测房价、根据广告投入估计销量。线性回归提供了可解释的基线：每个系数明确表示对应特征变化一个单位时，预测值的线性变化量（其他特征保持不变）。

## 直观理解

一元线性回归是在散点之间寻找一条最合适的直线。多元情形把直线推广为高维空间中的超平面。所谓“最合适”必须由损失函数定义，常见选择是让残差平方和最小。

## 正式定义

对特征 [[向量]] $\mathbf{x}\in\mathbb{R}^n$，模型为：

$$
\hat y=\mathbf{w}^{\mathsf T}\mathbf{x}+b
$$

其中 $\mathbf{w}$ 是权重，$b$ 是截距，$\hat y$ 是预测值。对 $m$ 个样本，使用均方误差时的目标函数为：

$$
J(\mathbf{w},b)=\frac{1}{m}\sum_{i=1}^{m}(\mathbf{w}^{\mathsf T}\mathbf{x}^{(i)}+b-y^{(i)})^2
$$

## 核心组成

1. 特征：模型可以使用的输入信息。
2. 权重：每个特征在线性组合中的系数。
3. 截距：所有特征为零时的基础预测值。
4. 残差：真实值与预测值之差。
5. 损失函数：汇总全部残差，评价参数质量。
6. 优化方法：寻找使损失尽量小的参数。

## 工作原理

```mermaid
flowchart LR
  A[输入特征 x] --> B[线性组合]
  C[权重 w 与截距 b] --> B
  B --> D[预测值 y_hat]
  D --> E[与真实值比较]
  E --> F[均方误差]
  F --> G[更新参数]
  G --> C
```

## 完整计算示例

用 $\hat y=wx+b$ 拟合两个样本 $(1,3)$、$(2,5)$。这两个点恰好在直线 $y=2x+1$ 上，因此取 $w=2,b=1$：

$$
\hat y_1=2\times1+1=3,\qquad \hat y_2=2\times2+1=5
$$

两个残差均为零，所以 MSE 为零。若取 $w=1,b=1$，预测为 $(2,3)$，则：

$$
\operatorname{MSE}=\frac{(2-3)^2+(3-5)^2}{2}=2.5
$$

## 梯度下降求解

对一元模型，MSE 关于 $w$ 和 $b$ 的梯度为：

$$
\frac{\partial J}{\partial w}=\frac{2}{m}\sum_{i=1}^{m}(\hat y_i-y_i)x_i,
\qquad
\frac{\partial J}{\partial b}=\frac{2}{m}\sum_{i=1}^{m}(\hat y_i-y_i)
$$

然后按照 [[梯度下降]] 的更新式重复调整两个参数。

## 代码或工程中的表现

```python
from sklearn.linear_model import LinearRegression

X = [[1], [2], [3]]
y = [3, 5, 7]

model = LinearRegression().fit(X, y)
print(model.coef_, model.intercept_)  # 约为 [2.] 和 1.0
```

## 实际应用

- 预测连续数值并建立可解释基线。
- 分析特征与目标之间的条件线性关系。
- 在更复杂模型之前检查数据管道与评价方式。

## 与相似概念的区别

- 逻辑回归名称中有“回归”，但通常用于分类，并通过 sigmoid 把线性得分映射为概率。
- 多项式回归在线性模型中加入多项式特征；它对参数仍然是线性的。
- 岭回归在损失中加入 $L_2$ 正则项，以限制过大的权重。

## 常见误区

- “线性”指对参数线性，不要求原始特征与目标的图像永远是一条直线。
- 相关关系和可预测性不自动构成因果关系。
- 高训练拟合度不保证泛化能力，也不证明假设正确。

## 边界与限制

线性回归依赖关系近似线性、误差结构适合所用推断方式等条件。强共线性会使系数不稳定；异常值可能显著影响平方损失；外推到训练数据范围之外风险较高。
