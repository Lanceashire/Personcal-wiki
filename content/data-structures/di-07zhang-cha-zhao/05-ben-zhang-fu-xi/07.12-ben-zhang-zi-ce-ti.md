## 八、代码改错

### 1. 顺序查找

```c
int search(const int arr[], int n, int target) {
    for (int i = 0; i <= n; i++) {
        if (arr[i] == target) {
            return i;
        }
    }

    return 0;
}
```

找出两个问题。

### 2. 折半查找

```c
int binarySearch(const int arr[], int n, int target) {
    int low = 0;
    int high = n - 1;

    while (low < high) {
        int mid = (low + high) / 2;

        if (arr[mid] == target) {
            return mid;
        } else if (arr[mid] < target) {
            low = mid;
        } else {
            high = mid;
        }
    }

    return -1;
}
```

找出可能导致漏查或死循环的问题。

### 3. 链表查找

```c
Node *search(Node *head, int target) {
    Node *p = head;

    while (p->data != target) {
        p = p->next;
    }

    return p;
}
```

指出空链表和目标不存在时的问题。
