```text
数据结构研究：

有穷性、确定性、可行性、输入、输出

```

## 2. 高频对比

| 概念 A | 概念 B | 核心区别 |
|---|---|---|
| 数据元素 | 数据项 | 元素是整体处理单位，数据项是组成元素的更小单位 |
| 逻辑结构 | 存储结构 | 抽象关系 vs 计算机中的具体表示 |
| 顺序存储 | 链式存储 | 连续地址与地址公式 vs 指针连接 |
| 算法基本特性 | 好算法评价标准 | 算法能否成立 vs 算法质量如何 |
| 时间复杂度 | 实际运行时间 | 增长数量级 vs 具体机器上的秒数 |
| 输入空间 | 辅助空间 | 问题本身的数据 vs 算法额外使用的空间 |

## 3. 选择与判断题

1. 数据结构只研究数据在内存中的存储方式。（ ）
2. 一个算法可以没有输入，但不能没有输出。（ ）
3. 两层嵌套循环的时间复杂度一定是 $O(n^2)$。（ ）
4. 无序数组按值查找的最坏复杂度通常是 $O(n)$。（ ）
5. 递归算法没有创建数组，因此空间复杂度一定是 $O(1)$。（ ）
6. 线性表是一种逻辑结构，数组是实现它的常用工具。（ ）

答案：

```text
1. 错
2. 对
3. 错
4. 对
5. 错
6. 对
```

## 4. 复杂度计算

### 题 1

```c
for (int i = 0; i < n; i++) {
    for (int j = 0; j < i; j++) {
        operation();
    }
}
```

$$
0+1+\cdots+(n-1)=\frac{n(n-1)}{2}=O(n^2)
$$

### 题 2

```c
for (int i = 1; i <= n; i *= 3) {
    operation();
}
```

执行次数约为 $\log_3 n$：

$$
O(\log n)
$$

### 题 3

```c
for (int i = 1; i <= n; i *= 2) {
    for (int j = 0; j < i; j++) {
        operation();
    }
}
```

$$
1+2+4+\cdots+n=O(n)
$$
