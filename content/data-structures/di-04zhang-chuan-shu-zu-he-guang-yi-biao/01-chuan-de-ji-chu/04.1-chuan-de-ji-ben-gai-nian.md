## 条目概览

串是由零个或多个字符组成的有限序列。串的逻辑特点是元素都是字符，常见操作以整个串或连续子串为对象。

> [!summary]
> 串是由零个或多个字符组成的有限序列。串的逻辑特点是元素都是字符，常见操作以整个串或连续子串为对象。

上级知识点：本章目录

## 1. 规范定义

串可记为：

$$
S=s_0s_1\cdots s_{n-1}
$$

其中：

- $s_i$ 是一个字符；
- $n$ 是串长；
- 本笔记默认代码下标从 `0` 开始。

例如：

```text
S = "DATA"
```

则：

```text
S[0] = 'D'
S[1] = 'A'
S[2] = 'T'
S[3] = 'A'
length = 4
```

## 2. 空串、空格串与空字符

### 空串

长度为 `0`：

```c
""
```

### 空格串

由一个或多个空格字符组成：

```c
" "
```

它的长度不是 `0`。一个空格的 ASCII 编码通常为 `32`。

### C 字符串结束符

```c
'\0'
```

用于标记 C 风格字符串结束，本身不是逻辑字符串内容的一部分。

```c
char text[] = "abc";
```

实际数组：

```text
['a']['b']['c']['\0']
```

数组占 `4` 个字符单元，但逻辑串长为 `3`。

> [!warning] 易错点
> `""`、`" "`、`'\0'` 完全不同：前者是空串，中间是长度为 1 的空格串，后者是单个结束标记字符。

## 3. 主串、子串与位置

若：

```text
S = "datastructure"
```

其中连续的一段：

```text
"structure"
```

是 `S` 的子串，`S` 是它的主串。

子串必须连续。

```text
"dt"
```

虽然字符按先后出现在 `"data"` 中，但不连续，所以不是子串；它可以称为子序列。

## 4. 子串数量计算

长度为 $n$ 的串：

### 非空子串位置数

选择起点和终点：

$$
\frac{n(n+1)}{2}
$$

例如 $n=3$：

```text
"abc"
```

非空子串按位置计：

```text
"a", "b", "c", "ab", "bc", "abc"
```

共：

$$
\frac{3\times4}{2}=6
$$

若把空串也算作一个子串，则总数：

$$
\frac{n(n+1)}{2}+1
$$

注意这是按“出现位置”计算，不是不同内容的数量。对于 `"aaa"`，多个位置得到的 `"a"` 内容相同，但出现位置不同。

## 5. 串相等

两个串相等必须同时满足：

1. 长度相等；
2. 对应位置字符全部相等。

```text
"abc" == "abc"
"abc" != "abd"
"abc" != "abc "
```

## 6. 字典序比较

从左到右比较第一个不同字符：

```text
"abc" < "abd"
```

因为：

```text
'c' < 'd'
```

若较短串是较长串的前缀：

```text
"abc" < "abcd"
```

C 标准库 `strcmp` 的返回值只保证：

```text
< 0、== 0、> 0
```

不应依赖它一定返回 `-1` 或 `1`。

## 7. 串与字符数组

串是逻辑概念；字符数组是 C 中常用的存储方式。

```c
char text[10] = "abc";
```

数组容量为 `10`，当前 C 字符串长度为 `3`。

与顺序表类似，必须区分：

- 数组容量；
- 当前串长；
- 结束符位置。

## 8. 串与线性表

串可以看作数据元素类型固定为字符的线性表，但其常见操作有明显特点：

- 比较整个串；
- 连接两个串；
- 截取连续子串；
- 查找模式串；
- 替换子串。

因此教材通常把串作为独立章节研究。

## 9. 常见计算

### 例 1

```text
S = "ababa"
```

串长：

$$
|S|=5
$$

模式 `"aba"` 的出现位置：

```text
0 和 2
```

两个出现可以重叠。

### 例 2

长度为 5 的串有多少个非空子串位置？

$$
\frac{5\times6}{2}=15
$$

### 例 3

长度为 $n$ 的主串中，长度为 $m$ 的模式最多有多少个合法起点？

当 $n\ge m$ 时：

$$
n-m+1
$$

若 $n<m$，合法起点为 `0` 个。
