## 条目概览

空串 ""：长度0 空格串 " "：长度1 C结束符 '\0'：不计入逻辑串长

上级知识点：本章目录

## 1. 一页速记

```text
空串 ""：长度0
空格串 " "：长度1
C结束符 '\0'：不计入逻辑串长

子串：必须连续
长度n的非空子串位置数：n(n+1)/2

BF：
失配后候选起点+1
最坏 O(nm)

pi[i]：
P[0..i] 的最长相等真前后缀长度

KMP失配：
j = pi[j-1]
主串指针不回退

next0：
next0[0]=-1
next0[j]=pi[j-1]

nextval：
跳过与刚失配字符相同的候选模式字符
```

## 2. 必背换算

$$
next0[0]=-1
$$

$$
next0[j]=\pi[j-1],\quad j\ge1
$$

若位置改为 1 下标：

$$
next1[j+1]=next0[j]+1
$$

## 3. 数组计算题

### 题 1：`ABABC`

```text
pi     = [0, 0, 1, 2, 0]
next0  = [-1, 0, 0, 1, 2]
next1  = [0, 1, 1, 2, 3]
```

### 题 2：`ababaca`

```text
pi      = [0, 0, 1, 2, 3, 0, 1]
next0   = [-1, 0, 0, 1, 2, 3, 0]
nextval = [-1, 0, -1, 0, -1, 3, -1]
```

### 题 3：`aaaab`

```text
pi      = [0, 1, 2, 3, 0]
next0   = [-1, 0, 1, 2, 3]
nextval = [-1, -1, -1, -1, 3]
```

### 题 4：`aabaaab`

```text
pi = [0, 1, 0, 1, 2, 2, 3]
```

## 4. 匹配计算

```text
T = "abababca"
P = "ababca"
```

```text
pi = [0,0,1,2,0,1]
```

前四个字符匹配后，在：

```text
T[4]='a'
P[4]='c'
```

失配：

```text
j=pi[3]=2
```

主串 `i=4` 不动，再与 `P[2]='a'` 比较。最终成功起点：

$$
2
$$

## 5. 判断题

1. 空格串是空串。（ ）
2. 子序列一定是子串。（ ）
3. `pi[i]` 可以等于 `i+1`。（ ）
4. KMP 失配时主串指针一定回到本轮起点后一位。（ ）
5. `next0[j]` 与 `pi[j]` 永远相等。（ ）
6. `nextval` 改变 KMP 的最坏渐近复杂度。（ ）
7. KMP 可以查找重叠匹配。（ ）
8. 长度为 $n$ 的串有 $n(n+1)/2$ 个非空子串位置。（ ）

答案：

```text
1错 2错 3错 4错
5错 6错 7对 8对
```

## 7. 手算步骤

计算 `pi`：

```text
1. 写模式下标和字符
2. pi[0]=0
3. k=pi[i-1]
4. 失配则 k=pi[k-1]
5. 相等则 k++
6. pi[i]=k
```

计算教材 `next`：

```text
1. 先确认0/1下标
2. 确认首项是-1还是0
3. 可先求pi
4. 按定义错位换算
5. 用搜索语句反查定义是否匹配
```
