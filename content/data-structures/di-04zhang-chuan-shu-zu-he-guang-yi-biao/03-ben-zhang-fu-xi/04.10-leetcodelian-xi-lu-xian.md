## 第一阶段：字符串基础

### [58. 最后一个单词的长度](https://leetcode.cn/problems/length-of-last-word/)

训练：

- 字符串边界；
- 跳过尾部空格；
- 下标从末尾向前移动。

### [344. 反转字符串](https://leetcode.cn/problems/reverse-string/)

训练：

- 字符数组；
- 双指针；
- 原地交换；
- 空串与单字符。

### [541. 反转字符串 II](https://leetcode.cn/problems/reverse-string-ii/)

训练：

- 分段边界；
- 最后不足 `k` 或 `2k` 个字符的处理；
- 区间 `[left, right]` 计算。

### [796. 旋转字符串](https://leetcode.cn/problems/rotate-string/)

关键结论：

```text
goal 是 s 的旋转
⇔ 长度相等且 goal 是 s+s 的子串
```

可以使用库函数，也可使用 KMP 完成子串查找。

## 第三阶段：前缀函数与周期

### [459. 重复的子字符串](https://leetcode.cn/problems/repeated-substring-pattern/)

使用：

```text
p = n - pi[n-1]
```

若：

```text
pi[n-1] > 0 且 n % p == 0
```

则串可由长度为 `p` 的子串重复构成。

例：

```text
"ababab"
pi最后值=4
p=2
6%2=0
```

### [1392. 最长快乐前缀](https://leetcode.cn/problems/longest-happy-prefix/)

答案长度：

```text
pi[n-1]
```

这是理解 `pi` 定义最直接的一题。

### [214. 最短回文串](https://leetcode.cn/problems/shortest-palindrome/)

进阶构造：

```text
s + "#" + reverse(s)
```

对组合串求前缀函数，最后值对应原串最长回文前缀长度。

## 第四阶段：多模式与字符串算法扩展

KMP 解决单模式匹配。后续可了解：

- Rabin-Karp：滚动哈希；
- Z Algorithm：Z 函数；
- Trie：前缀集合；
- Aho-Corasick：多模式匹配；
- 后缀数组、后缀自动机。

这些不属于本轮考试基础，但有助于建立字符串算法地图。
