## 条目概览

KMP 的计算题通常考四类：求 pi/next/nextval、写匹配轨迹、计算成功位置、比较朴素算法与 KMP 的字符比较次数。

## 1. 复杂度推导

模式预处理：

$$
O(m)
$$

主串扫描：

$$
O(n)
$$

总时间：

$$
O(n+m)
$$

辅助空间：

$$
O(m)
$$

## 2. 为什么嵌套 `while` 仍是线性

构造 `pi` 时：

- `i` 从 `1` 到 `m-1`，只增不减；
- `k` 成功时增加；
- 失配时沿已知边界严格缩短。

搜索时：

- 主串 `i` 只向右；
- `j` 增加总量受 `i` 推进限制；
- 每次回退都使 `j` 变小。

可使用势能直觉理解：`j` 每增加一次，未来最多被对应地减少一次，总操作量为线性级别。

## 3. 综合计算一

模式：

```text
P = "aabaaab"
```

求 `pi`。

| `i` | 前缀 | 最长边界 | `pi[i]` |
|---:|---|---|---:|
| 0 | `a` | 无 | 0 |
| 1 | `aa` | `a` | 1 |
| 2 | `aab` | 无 | 0 |
| 3 | `aaba` | `a` | 1 |
| 4 | `aabaa` | `aa` | 2 |
| 5 | `aabaaa` | 回退后 `aa` | 2 |
| 6 | `aabaaab` | `aab` | 3 |

结果：

```text
pi = [0, 1, 0, 1, 2, 2, 3]
```

特别看 `i=5`：

```text
k=pi[4]=2
P[5]='a' != P[2]='b'
k=pi[1]=1
P[5]='a' == P[1]='a'
k++ => 2
```

所以不是 `3`，而是 `2`。

## 4. 综合计算二：换算数组

已知：

```text
pi = [0, 1, 0, 1, 2, 2, 3]
```

则：

```text
next0[0] = -1
next0[j] = pi[j-1]
```

得到：

```text
next0 = [-1, 0, 1, 0, 1, 2, 2]
```

教材 1 下标：

```text
next1 = [0, 1, 2, 1, 2, 3, 3]
```

## 5. 综合计算三：匹配轨迹

```text
T = "abababca"
P = "ababca"
pi = [0,0,1,2,0,1]
```

| 主串下标 `i` | 主串字符 | 匹配长度 `j` | 动作 |
|---:|---|---:|---|
| 0 | a | 0 | 相等，`j=1` |
| 1 | b | 1 | 相等，`j=2` |
| 2 | a | 2 | 相等，`j=3` |
| 3 | b | 3 | 相等，`j=4` |
| 4 | a | 4 | 与 `P[4]=c` 失配，`j=pi[3]=2` |
| 4 | a | 2 | 相等，`j=3` |
| 5 | b | 3 | 相等，`j=4` |
| 6 | c | 4 | 相等，`j=5` |
| 7 | a | 5 | 相等，`j=6=m` |

成功起点：

$$
7-6+1=2
$$

## 6. 综合计算四：重叠匹配

```text
T = "aaaaa"
P = "aaa"
pi = [0,1,2]
```

第一次成功结束于 `i=2`：

```text
起点 0
```

成功后：

```c
j = pi[2] = 2;
```

继续：

- `i=3` 成功，起点 `1`；
- 回退 `j=2`；
- `i=4` 成功，起点 `2`。

结果：

```text
0, 1, 2
```

## 7. 比较次数

### 朴素算法

最坏上界：

$$
(n-m+1)m
$$

### KMP

字符比较次数通常不超过某个与 $n+m$ 成正比的常数倍，渐近为：

$$
O(n+m)
$$

考试若要求精确比较次数，必须按照给定主串、模式串和具体 `next` 定义逐步记录，不能仅写复杂度。

## 8. 周期计算

对长度为 $m$ 的模式串，若：

$$
p=m-\pi[m-1]
$$

且：

$$
m\bmod p=0
$$

则 $p$ 是一个最小周期候选。

例如：

```text
P = "ababab"
pi最后值 = 4
p = 6 - 4 = 2
6 % 2 = 0
```

最小重复单位：

```text
"ab"
```

## 9. 最长快乐前缀

整个串的最长相等真前后缀长度：

```c
pi[m - 1]
```

内容：

```text
pattern[0 .. pi[m-1)-1]
```

这是 KMP 预处理的直接应用。
