## 条目概览

遇到左括号时暂存，遇到右括号时必须与最近尚未匹配的左括号配对。“最近尚未完成”正好对应栈的后进先出规则。

## 1. 问题定义

判断字符串中的：

```text
()  []  {}
```

是否满足：

1. 每个右括号都有对应左括号；
2. 类型相同；
3. 嵌套次序正确；
4. 最终没有未匹配左括号。

合法：

```text
()[]{}
([{}])
{[()()]}
```

非法：

```text
(]
([)]
(()
())
```

## 2. 为什么使用栈

扫描：

```text
([{}])
```

左括号入栈：

```text
(
( [
( [ {
```

遇到 `}` 时，需要匹配最近的 `{`；遇到 `]` 时匹配最近的 `[`。

这正是：

```text
最后进入的左括号最先被匹配
```

## 3. 算法不变量

扫描到任意位置时，栈中保存：

> 已经出现、但尚未被右括号匹配的左括号，栈顶是最近的一个。

处理规则：

1. 左括号：入栈；
2. 右括号：
   - 栈空：失败；
   - 栈顶类型不匹配：失败；
   - 匹配：弹出栈顶；
3. 扫描结束：
   - 栈空：成功；
   - 非空：仍有左括号未匹配，失败。

## 4. C 语言实现

```c
#include <string.h>

int isLeftBracket(char ch) {
    return ch == '(' || ch == '[' || ch == '{';
}

int isMatched(char left, char right) {
    return
        (left == '(' && right == ')') ||
        (left == '[' && right == ']') ||
        (left == '{' && right == '}');
}

int isValidBrackets(const char *text) {
    if (text == NULL) {
        return 0;
    }

    int length = (int)strlen(text);
    char *stack = malloc((size_t)length * sizeof(char));

    if (stack == NULL && length > 0) {
        return 0;
    }

    int top = -1;

    for (int i = 0; i < length; i++) {
        char ch = text[i];

        if (isLeftBracket(ch)) {
            stack[++top] = ch;
        } else if (
            ch == ')' ||
            ch == ']' ||
            ch == '}'
        ) {
            if (
                top < 0 ||
                !isMatched(stack[top], ch)
            ) {
                free(stack);
                return 0;
            }

            top--;
        }
    }

    int valid = (top == -1);
    free(stack);
    return valid;
}
```

## 5. 三类失败必须全部检查

### 5.1 右括号过早出现

```text
")
```

遇到右括号时栈空，失败。

### 5.2 类型不匹配

```text
(]
```

栈顶是 `(`，当前是 `]`，失败。

### 5.3 左括号剩余

```text
(()
```

扫描结束后栈非空，失败。

> [!warning] 易错点
> 只在遇到右括号时检查还不够。扫描结束后必须再次检查栈是否为空。

## 6. 逐步示例

输入：

```text
{[()]}
```

| 当前字符 | 操作 | 栈内容（底→顶） |
|---|---|---|
| `{` | 入栈 | `{` |
| `[` | 入栈 | `{ [` |
| `(` | 入栈 | `{ [ (` |
| `)` | 匹配并出栈 | `{ [` |
| `]` | 匹配并出栈 | `{` |
| `}` | 匹配并出栈 | 空 |

最终栈空，合法。

## 7. 复杂度

长度为 $n$ 的字符串只扫描一次：

$$
T(n)=O(n)
$$

最坏情况下全是左括号，栈保存 $n$ 个字符：

$$
S(n)=O(n)
$$

### 推荐解题顺序

1. 先写只处理 `()` 的版本；
2. 扩展到 `[]` 和 `{}`；
3. 加入奇数长度提前失败；
4. 用数组栈重写；
5. 再尝试“右括号映射到左括号”的写法。

### 不建议直接背的技巧

有些解法遇到左括号时压入“期望的右括号”。这种写法很简洁，但应先理解标准版本中的匹配不变量。
