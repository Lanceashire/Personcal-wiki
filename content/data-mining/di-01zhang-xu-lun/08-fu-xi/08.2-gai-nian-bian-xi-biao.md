## 条目概览

概念辨析表是数据挖掘知识体系中的一个条目，本文整理其核心概念、原理、示例与应用边界。

| 易混概念 | 区别 |
|---|---|
| 查询 vs 数据挖掘 | 查询执行已知条件；数据挖掘发现未知模式 |
| KDD vs 数据挖掘 | KDD 是完整知识发现流程；数据挖掘是其中核心步骤 |
| 数据多 vs 维度高 | 数据多通常指对象数量多；维度高指属性数量多 |
| 分类 vs 回归 | 分类输出离散类别；回归输出连续数值 |
| 分类 vs 聚类 | 分类通常有标签；聚类通常没有标签 |
| 关联 vs 因果 | 共同出现不能直接证明因果 |
| 异常 vs 错误 | 异常可能是真实但少见的现象 |
| 机器学习 vs 数据挖掘 | 机器学习关注学习方法；数据挖掘还关注预处理、评估和应用 |
| 随机样本 vs 机会性样本 | 随机样本按随机机制抽取；机会性样本来自日常运行 |
| 数据驱动 vs 理论指导 | 数据驱动发现模式；理论帮助解释和约束 |
