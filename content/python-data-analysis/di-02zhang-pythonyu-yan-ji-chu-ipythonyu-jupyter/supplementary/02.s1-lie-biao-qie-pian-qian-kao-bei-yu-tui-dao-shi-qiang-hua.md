## 1. 概念概览

本节是第二章的对话驱动补充。教材第二章讲了 Python 语言基础，但对列表切片赋值、浅拷贝、列表方法和推导式不会像期末复习那样集中训练。由于这些知识点在学习对话中反复出现，本节把它们整理成可独立复习的强化专题。

## 2. 学习重点

- 浅拷贝只复制外层容器，内部对象仍可能共享。
- 切片读取和切片赋值不是同一类动作。
- 普通切片赋值和扩展切片赋值规则不同。
- 列表方法常原地修改并返回 `None`。
- 推导式要按“遍历 -> 筛选 -> 转换”阅读。

## 3. 规范定义

| 主题 | 核心规则 |
| --- | --- |
| 浅拷贝 | 新建外层容器，元素仍引用原对象 |
| 切片读取 | 返回新外层列表，不修改原列表 |
| 普通切片赋值 | `step == 1`，连续片段替换，长度可变 |
| 扩展切片赋值 | `step != 1`，逐个位置替换，数量必须一致 |
| `append` | 整体追加一个元素 |
| `extend` | 遍历右侧对象并逐个追加 |
| 推导式筛选 `if` | 决定哪些元素进入结果 |
| 条件表达式 `if ... else` | 决定元素如何转换 |

## 4. 直觉解释

切片放在等号右侧时，是“取一份结果”；切片放在等号左侧时，是“指定要修改原列表的哪些位置”。

```python
b = a[:]       # 读取切片：创建新外层列表
a[:] = [100]   # 切片赋值：原地修改原列表内容
a = [100]      # 变量重绑定：让 a 指向新列表
```

这三行的对象关系完全不同。

## 5. 原理与机制

### 5.1 浅拷贝与嵌套列表

```python
a = [[1, 2], [3, 4]]
b = a.copy()
b[0].append(5)

print(a)
print(b)
print(a is b)
print(a[0] is b[0])
```

输出：

```python
[[1, 2, 5], [3, 4]]
[[1, 2, 5], [3, 4]]
False
True
```

`a` 和 `b` 的外层列表不同，但第一个子列表仍是同一个对象。

### 5.2 替换引用与修改共享对象

```python
b[0] = [100]        # 替换 b 外层列表中的一个引用
b[0].append(100)    # 修改 b[0] 指向的子列表对象
```

第一种通常不会影响 `a[0]`；第二种如果子列表共享，就会影响。

### 5.3 切片赋值

普通切片赋值：

```python
a = [0, 1, 2, 3, 4, 5, 6]
a[2:5] = [10, 20]
print(a)  # [0, 1, 10, 20, 5, 6]
```

扩展切片赋值：

```python
a = [0, 1, 2, 3, 4]
a[::2] = [100, 200, 300]
print(a)  # [100, 1, 200, 3, 300]
```

错误示例：

```python
a = [0, 1, 2, 3, 4]
a[::2] = [100]
```

`a[::2]` 选中下标 `0, 2, 4`，共三个位置，右侧只有一个值，因此报 `ValueError`。

## 6. 关键步骤、分类或公式

判断切片赋值是否合法：

1. 找出左侧切片的 `step`。
2. 若 `step` 省略或等于 `1`，这是整段替换，右侧长度可以不同。
3. 若 `step != 1`，这是扩展切片赋值，右侧元素数量必须等于左侧选中位置数量。

可以记成：

$$
\text{扩展切片赋值合法} \Longleftrightarrow
\lvert \text{右侧元素} \rvert = \lvert \text{左侧选中位置} \rvert
$$

## 7. 例题与具体案例

### 例题 1：列表方法返回值

```python
a = [1, 2]
result = a.append([3, 4])
a.extend("56")
a.insert(-1, 99)

print(a)
print(result)
```

输出：

```python
[1, 2, [3, 4], '5', 99, '6']
None
```

`append` 把 `[3, 4]` 作为一个整体元素加入；`extend("56")` 把字符串逐字符加入；`insert(-1, 99)` 插到最后一个元素之前。

### 例题 2：推导式清洗

```python
data = [[" Python ", None, ""], [" NumPy", "  ", "Pandas"]]

result = [
    x.strip().upper()
    for line in data
    for x in line
    if x is not None and x.strip() and len(x.strip()) > 2
]

print(result)
```

输出：

```python
["PYTHON", "NUMPY", "PANDAS"]
```

## 8. 易错点与边界情况

> [!warning] 对话中暴露的易错点
> `a[::] = [100]` 可以把列表原地变成 `[100]`，但 `a[::2] = [100]` 不一定可以。核心差异是 `step == 1` 还是 `step != 1`。

其他易错点：

- `remove(x)` 删除第一个值为 `x` 的元素，返回 `None`。
- `pop(i)` 删除下标 `i` 的元素，返回被删除的值。
- `del a[i]` 是语句，不返回值。
- `a[::-1]` 是读取反向新列表，不修改原列表。
- `a[::-1] = [...]` 是扩展切片赋值，右侧数量必须一致。
