## 条目概览

教材介绍 Python 的标量类型，包括数字、字符串、布尔值、None 和日期时间等。对数据分析来说，字符串与缺失值尤其重要：原始表格数据里常常混有文本、空白、缺失、异常编码和数值。

## 1. 概念概览

教材介绍 Python 的标量类型，包括数字、字符串、布尔值、`None` 和日期时间等。对数据分析来说，字符串与缺失值尤其重要：原始表格数据里常常混有文本、空白、缺失、异常编码和数值。

本节重点放在字符串和 `None`，因为对话中已经暴露出这部分的典型误区：`None` 不是字符串，不能调用 `.strip()`；空字符串可以调用 `.strip()`，但清洗后可能仍然是假值。

## 2. 学习重点

- 字符串是不可变对象，字符串方法返回新字符串。
- `None` 表示空值或无值，是独立对象，不是空字符串。
- 空字符串 `""`、纯空格字符串 `"   "`、`None` 三者不同。
- `and` 的短路求值可以用于安全过滤。
- 数据清洗时不能把 `0`、`None`、空字符串混为一谈。

## 3. 规范定义

| 值 | 类型 | 条件判断中真假 | 能否调用 `.strip()` |
| --- | --- | --- | --- |
| `None` | `NoneType` | False | 不能 |
| `""` | `str` | False | 能 |
| `"   "` | `str` | True | 能，结果为 `""` |
| `" Python "` | `str` | True | 能，结果为 `"Python"` |
| `0` | `int` | False | 不能 |

## 4. 直觉解释

`None` 表示“没有这个值”；空字符串表示“有一个字符串，只是长度为 0”；纯空格字符串表示“有字符串，但内容都是空白”。这三者在数据清洗中可能代表不同业务含义。

## 5. 原理与机制

字符串清洗示例：

```python
data = ["  Python  ", "", "   ", " NumPy "]
result = [x.strip().lower() for x in data if x.strip()]

print(result)
```

输出：

```python
["python", "numpy"]
```

这里不是 `strip()` 从列表中删除元素，而是 `if x.strip()` 过滤掉清洗后为空字符串的元素。

含 `None` 时必须先判断：

```python
data = ["  Python  ", None, "", " NumPy "]

result = [
    x.strip().lower()
    for x in data
    if x is not None and x.strip()
]
```

`and` 从左到右判断。若 `x is not None` 为 `False`，后面的 `x.strip()` 不会执行。

## 6. 关键步骤、分类或公式

安全清洗文本字段：

```text
先排除 None
  -> 再 strip 去两端空白
  -> 再判断是否为空
  -> 再做 lower / upper / 其他转换
```

对应代码：

```python
cleaned = [
    x.strip().lower()
    for x in values
    if x is not None and x.strip()
]
```

## 7. 例题与具体案例

### 例题：长度筛选

要求：保留非 `None`、去空白后长度大于 3 的字符串，并转小写。

```python
data = [[" Python ", None], [" NumPy", "  R ", ""]]

result = [
    x.strip().lower()
    for line in data
    for x in line
    if x is not None and x.strip() and len(x.strip()) > 3
]

print(result)
```

输出：

```python
["python", "numpy"]
```

## 8. 易错点与边界情况

> [!warning] 对话中暴露的易错点
> `x != None` 虽然有时能运行，但判断空值更推荐 `x is not None`。更重要的是，`x is not None` 不能过滤空字符串；过滤空白字符串还需要 `x.strip()`。

错误写法：

```python
if x.strip() and x is not None
```

当 `x = None` 时，会先执行 `None.strip()`，直接报 `AttributeError`。

另一个常见错误：

```python
if x > 3
```

这不能判断字符串长度。应写：

```python
if len(x.strip()) > 3
```
