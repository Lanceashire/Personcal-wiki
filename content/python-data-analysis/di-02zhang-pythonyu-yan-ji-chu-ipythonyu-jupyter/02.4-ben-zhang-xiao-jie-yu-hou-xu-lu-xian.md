## 1. 概念概览

第二章的目标是让 Python 语言基础足够稳定，可以进入数据结构、NumPy 和 pandas。解释器、IPython、Jupyter 是运行和探索环境；变量、对象、可变性、字符串、`None` 和控制流是写出正确数据清洗逻辑的基础。

## 2. 学习重点

- 能用解释器和 Notebook 验证代码。
- 能判断变量名是否共享同一个对象。
- 能区分原地修改和重新绑定。
- 能安全处理字符串和 `None`。
- 能读懂基本控制流和推导式。

## 5. 原理与机制

后续学习迁移关系：

```text
Python 列表切片 -> NumPy 数组切片 -> pandas 行列选择
Python 字符串方法 -> pandas Series.str 方法
Python None 判断 -> pandas 缺失值处理
Python for / if -> pandas 布尔筛选和分组逻辑
Python 对象引用 -> DataFrame 复制和原地修改风险
```

## 7. 例题与具体案例

若能解释下面代码，就说明本章核心已经较稳：

```python
data = [[" Python ", None], ["", " NumPy "]]

result = [
    x.strip().lower()
    for row in data
    for x in row
    if x is not None and x.strip()
]

print(result)
```

输出：

```python
["python", "numpy"]
```

该题同时考查嵌套循环、字符串方法、`None` 判断、短路求值和推导式。
