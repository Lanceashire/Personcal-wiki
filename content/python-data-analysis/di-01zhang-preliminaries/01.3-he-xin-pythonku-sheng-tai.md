## 条目概览

教材第一章列出数据分析中最重要的一组 Python 库。它们不是互相替代的关系，而是分工协作：

## 1. 概念概览

教材第一章列出数据分析中最重要的一组 Python 库。它们不是互相替代的关系，而是分工协作：

- NumPy 负责高效数组和数值计算；
- pandas 负责表格数据分析；
- Matplotlib 负责基础可视化；
- IPython 和 Jupyter 负责交互式探索；
- SciPy 提供科学计算算法；
- scikit-learn 提供机器学习工具；
- statsmodels 提供统计建模。

## 2. 学习重点

第一章不要求马上掌握每个库 API，但必须知道：

1. 每个库解决哪类问题；
2. 后续章节什么时候会用到它；
3. Python 基础薄弱点会如何影响这些库的学习。

## 3. 规范定义

| 库 | 核心对象或功能 | 典型用途 |
| --- | --- | --- |
| NumPy | `ndarray` | 数组、矩阵、向量化计算、随机数、线性代数基础 |
| pandas | `Series`、`DataFrame` | 表格数据读取、清洗、筛选、分组、聚合、合并 |
| Matplotlib | `Figure`、`Axes` | 折线图、柱状图、散点图、图表定制 |
| IPython / Jupyter | Notebook、交互式 shell | 探索式分析、边运行边记录 |
| SciPy | 科学计算函数 | 优化、积分、统计、信号处理等 |
| scikit-learn | 估计器 API | 分类、回归、聚类、模型评估 |
| statsmodels | 统计模型 | 回归、时间序列、统计检验 |

## 4. 直觉解释

可以把 Python 数据分析库生态理解为一张工作台：

```text
NumPy：底层数值材料
pandas：表格操作台
Matplotlib：画图板
Jupyter：实验记录本
SciPy：科学计算工具箱
scikit-learn：机器学习工具箱
statsmodels：统计推断工具箱
```

## 5. 原理与机制

### 5.1 NumPy 是数值计算底座

NumPy 的数组要求数据类型更统一，因此能进行高效批量运算。后续 pandas 的许多能力也建立在 NumPy 风格的数组计算之上。

### 5.2 pandas 面向表格数据

pandas 把数据组织为带标签的行列结构，更接近 CSV、Excel、数据库表。它解决的是现实分析中最常见的问题：字段不干净、类型混乱、需要筛选分组、多个表要合并。

### 5.3 Matplotlib 面向表达

分析结果不画出来，很多模式难以发现。Matplotlib 是 Python 可视化生态中的基础库，很多高级绘图库也会建立在它之上或与它协作。

### 5.4 Jupyter 面向探索过程

Jupyter Notebook 适合逐步运行代码、观察中间结果、记录解释文字。但正式项目仍需要注意代码组织，避免把所有逻辑堆在零散单元格里。

## 6. 关键步骤、分类或公式

库的选择可以按数据分析流程判断：

| 阶段 | 主要问题 | 常用库 |

| 检查 | 字段、类型、缺失、重复 | pandas |

## 7. 例题与具体案例

### 例题：选择合适工具

场景：有一个 `students.csv`，字段包括 `name`、`score`、`class`。需要去掉无效姓名，筛选有效成绩，按班级求平均分并画柱状图。

合理工具链：

```python
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("students.csv")
df = df[df["name"].notna() & df["name"].str.strip().ne("")]
df = df[df["score"].notna() & (df["score"] > 0)]

avg = df.groupby("class")["score"].mean()
avg.plot(kind="bar")
plt.show()
```

这段代码背后仍然依赖基础 Python 思维：空值判断、字符串清洗、布尔筛选、分组聚合。

## 8. 易错点与边界情况

> [!warning] 易错点
> 不能把库名背熟就当作掌握数据分析。真正重要的是知道每个库在流程中的位置，以及什么时候该用基础 Python，什么时候该用 pandas 或 NumPy。

常见边界情况：

- 小列表练习适合基础 Python；大规模表格应尽早转向 pandas。
- 数值数组计算优先考虑 NumPy 向量化，而不是 Python `for` 循环。
- 画图前要先明确图表回答的问题。

## 9. 与相关知识点的联系

- 对话中的列表、切片、推导式会帮助理解 NumPy 索引和数组筛选。
- `strip()`、`lower()` 等字符串方法会迁移到 pandas 的 `.str` 字符串访问器。
- `None` 与空字符串的区别，会扩展为 pandas 中 `NaN`、空文本和异常编码值的区别。
